# Flujo general del DM y formato de salida

Base canonica: `00-PuntosFijosCanonicos.md`.
Este archivo orquesta el uso de los demas LLM.
No redefine mecanicas base.

## Orden de consulta recomendado
1. Arranque de escena: `03-PrearranqueEscenaObligatorio.md` (PF-09)
2. Persistencia y memoria minima: `04-GobernanzaMemoriaPersistencia.md`
3. Decision base de accion: `01-MecanicaBaseResolucion.md` (PF-01..PF-04)
4. Si hay duda bloqueante, resolver primero con el usuario (ver `04-GobernanzaMemoriaPersistencia.md`).
5. Si aplica:
- Individual vs grupal: `20-TiradasIndividualVsGrupal.md` (PF-05)
- Accion con apoyo: `21-TiradasConApoyo.md` (PF-06)
- PvP: `30-ResolucionPvP.md` (PF-07)
- Coordenadas y mapa: `11-CoordenadasTacticas.md` (PF-08)
6. Interpretacion final: `22-InterpretacionConsecuencias.md` (PF-10)

## Convencion practica de tirada
- Jugador reporta resultado natural.
- DM aplica modificadores segun personaje.
- DM narra consecuencia.

## Formato de respuesta del DM (PF-10)
1. Escena
2. Resultado
3. Estado actualizado
4. Tiradas requeridas
