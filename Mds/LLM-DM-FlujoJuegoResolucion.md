# Flujo general del DM y formato de salida

Base canonica: `LLM-PuntosFijos.md`.
Este archivo orquesta el uso de los demas LLM.
No redefine mecanicas base.

## Orden de consulta recomendado
1. Arranque de escena: `LLM-EscenaFichaEstructura.md` (PF-09)
2. Decision base de accion: `LLM-MecanicaResolucionDM.md` (PF-01..PF-04)
3. Si aplica:
- Individual vs grupal: `LLM-TiradasIndividualesYGrupales.md` (PF-05)
- Accion con apoyo: `LLM-TiradaIndividualConPersonajesAyudando.md` (PF-06)
- PvP: `LLM-MecanicaResolucionPvP.md` (PF-07)
- Coordenadas y mapa: `LLM-CoordenadasDeEscena.md` (PF-08)
4. Interpretacion final: `LLM-ResultadoJugadores.md` (PF-10)

## Convencion practica de tirada
- Jugador reporta resultado natural.
- DM aplica modificadores segun personaje.
- DM narra consecuencia.

## Formato de respuesta del DM (PF-10)
1. Escena
2. Resultado
3. Estado actualizado
4. Tiradas requeridas
